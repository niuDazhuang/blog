## [观察者模式](https://www.zhihu.com/question/23486749)

- 观察者需要直接订阅目标事件
- 在目标发出内容改变的事件后，直接接收事件并作出响应

```
╭─────────────╮  Fire Event  ╭──────────────╮
│             │─────────────>│              │
│   Subject   │              │   Observer   │
│             │<─────────────│              │
╰─────────────╯  Subscribe   ╰──────────────╯

```

## 发布订阅模式

- 发布者和订阅者之间存在一个发布通道
- 订阅者需要从事件通道订阅事件

```
╭─────────────╮                 ╭───────────────╮   Fire Event   ╭──────────────╮
│             │  Publish Event  │               │───────────────>│              │
│  Publisher  │────────────────>│ Event Channel │                │  Subscriber  │
│             │                 │               │<───────────────│              │
╰─────────────╯                 ╰───────────────╯    Subscribe   ╰──────────────╯
```
